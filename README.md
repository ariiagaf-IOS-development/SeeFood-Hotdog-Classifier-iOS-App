<h1 align="center">SeeFood – Hotdog Classifier iOS App</h1>

<p align="center">
  An iOS image classification app built with Swift, UIKit, CoreML, and Vision that detects whether a photo contains a hotdog.
</p>

## Features

- Image classification with CoreML
- Camera-based photo input
- Hotdog / Not Hotdog prediction
- Vision framework integration
- Simple UIKit interface

## Tech Stack

- Swift
- UIKit
- CoreML
- Vision
- Xcode

## What I Practiced

- Integrating a CoreML model into an iOS app
- Using Vision for image classification
- Working with `UIImagePickerController`
- Handling camera input and image processing
- Building a simple UIKit-based machine learning app

## Run the Project

1. Open the project in Xcode
2. Run it on a physical iPhone device
3. Allow camera access when prompted
